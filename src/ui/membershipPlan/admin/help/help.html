Gestión de Usuarios: Ayuda

Descripción general

Esta ventana permite realizar las operaciones de mantenimiento básico de los datos de los planes de membresía. El usuario podrá mediante los controles de los que dispone la ventana crear, modificar y eliminar los planes de la aplicación. Así mismo podrá generar un informe que contiene un listado de los planes con sus correspondientes datos. Además, puede realizar búsquedas por tres parámetros diferentes: nombre, precio y duración del planes.

Operaciones

Crear planes

Para crear un plan deberá introducir un identificador único que se utilizará como login y el nombre del usuario en los campos de entrada correspondientes.

  • Name: Nombre del plan (Tiene un límite de 25 caracteres).
  • Description: Descripción del plan (Tiene un límite de 150 caracteres).
  • Album limit: Limite de álbumes que puede tener un usuario con este plan (Tiene un límite de 6 caracteres y solamente permite números)
  • Duration: Duración del plan en meses (Tiene un límite de 6 caracteres y solamente permite números)
  • Price: Precio del plan (Tiene un límite de 6 caracteres y solamente permite números con decimales separados por un “.”
  • Shareable: Propiedad del plan que indica que el usuario puede compartir álbumes (marcada para sí, desmarcada para no)
Una vez realizadas las acciones anteriores, deberá pulsar el botón Create para crear el plan. Si no se produce ningún error, se creará el plan y aparecerá en la tabla de planes que se muestra en la ventana.

Modificar datos de los planes

Para modificar los datos de un usuario primero deberá seleccionar el usuario haciendo click en fila deseada de la tabla de usuarios. Los campos de entrada de la ventana se informarán con los datos del usuario seleccionado. Dichos campos se pueden utilizar para modificar los datos del usuario. Una vez realizadas las acciones anteriores, deberá pulsar el botón Modificar para finalizar la operación. Si no se produce ningún error, se modificarán los datos del usuario y los datos actualizados del usuario aparecerán en la tabla de usuarios que se muestra en la ventana.

Eliminar usuarios

Para eliminar un usuario deberá seleccionar el usuario a borrar haciendo click en fila deseada de la tabla de usuarios. A continuación, pulse en el botón Eliminar, y se le solicitará confirmación para eliminar el usuario. Si confirma el borrado y no se produce ningún error se eliminará el usuario y se actualizará la tabla de usuarios que se muestra en la ventana.

Imprimir listado de usuarios

Pulsando el botón Imprimir se abrirá una ventana donde puede ver un documento que contiene un listado de los usuarios que existen en la aplicación:



Mediante los botones de la parte superior de la ventana podrá imprimir o almacenar dicho informe.

Salir de la aplicación

Pulsando el botón Salir se cerrará la aplicación.
